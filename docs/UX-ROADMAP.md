# راهنمای بهبود UX — customer-site

> هدف: سایت کاملاً کاربرپسند با طراحی مدرن، واکنش‌گرا و دسترسی‌پذیر.

---

## وضعیت فعلی (خلاصه)

| بخش | وضعیت | نکات |
|-----|--------|------|
| Layout | پایه | هدر ساده، منو افقی، بدون منوی موبایل |
| صفحهٔ خانه | ساده | لیست متن‌محور، بدون Hero |
| وبلاگ | پایه | لیست ساده، بدون کارت/تصویر |
| فروشگاه | پایه | لیست ساده، بدون تصویر محصول |
| سبد خرید | پایه | جدول ساده، فرم checkout |
| تماس | پایه | فرم بدون استایل مناسب |
| فرم‌ها | ضعیف | label و input بدون کلاس، بدون فیدبک خطا |
| موبایل | محدود | منو در موبایل شکسته می‌شود |
| خطاها | پایه | صفحات 404/500 استاندارد Laravel |
